# Frozen Baseline Notes

This folder preserves the public source material from `JordanG2D/G2-Researcher_v1.0_FROZEN`.

The import keeps the original application structure and source files for historical review. It excludes generated caches, private run logs, and live deployment configuration that did not add useful archival context.

Environment files were not imported. The historical code still references model and sandbox environment variable names, but no credential file or secret value is included.

This code is preserved as an early baseline artifact. It should not be read as current G2D, Orion, or Black Box Lab production infrastructure.
